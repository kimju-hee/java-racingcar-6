# java-racingcar

## 구현 기능

- [ ] **1. 입력 받기**
    - 자동차 이름 입력 받기
      - 쉼표로 구분
      - **예외 처리**
        - 영어로만 이루어졌는지
        - 5자 이하인지
    - 횟수 입력 받기
      - **예외 처리**
        - 숫자인지
    - MAP 사용하여 자동차 별로 전진 칸 저장

<br>


- [ ] **2. 자동차 전진하기**
  - 횟수만큼 각 자동차 랜덤 돌리기
    - 4 이상일 경우 전진
  - 차수별로 실행 결과 출력하기

<br>

- [ ] **3. 우승자 문구 출력**
  - 여러 명일 경우 쉼표로 구분
